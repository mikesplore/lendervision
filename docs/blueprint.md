# **App Name**: LenderVision

## Core Features:

- Identity Verification: Use AI-powered liveness detection and ID verification to prevent fraud during borrower registration.
- Financial Data Connection: Allow borrowers to connect their financial data via M-Pesa, bank statements, or digital wallets. AI tool analyzes transaction history to determine loan eligibility.
- Application Status Dashboard: Provide borrowers with a clear, real-time view of their application status.
- Lender Dashboard - Incoming Queue: Display a table-based view of loan applicants with key metrics and AI-driven risk ratings.
- Applicant Deep-Dive - AI Assessment Hub: Provide lenders with a three-panel view including profile summary, AI scores, detailed analysis, and recommendation tools.
- Fraud Detection & Alerts: Provide a detailed list of possible fraudulent activity for manual review
- Database for storing and accessing user and assessment information.: Implements integration with Firestore for storing applicant data, loan details, and AI assessment results.

## Style Guidelines:

- Primary color: Trustworthy blue (#3498DB), reflecting reliability and stability. This is an adaptation of the requested blue and white color scheme. In HSL it is (202, 63%, 53%).
- Background color: Light gray (#ECF0F1), creating a clean and modern interface, to keep the focus on the app's information and processes. In HSL, that is hue 200, saturation 20%, lightness 89%.
- Accent color: Soft green (#2ECC71), signaling approval and positive outcomes, with the analogous color on the opposite side of the primary's hue in order to call attention to approved actions. In HSL it is (145, 61%, 53%).
- Body and headline font: 'PT Sans', a humanist sans-serif to combine a modern look and a little warmth or personality; suitable for headlines or body text.
- Use clear and recognizable icons to represent different data points, status indicators, and actions.
- Mobile-first design with a focus on clear progress indicators and intuitive navigation for borrowers. Desktop-focused 'command center' feel for lenders with high-density data visualization.
- Subtle animations to provide feedback on user actions and AI analysis progress.
